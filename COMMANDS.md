# BigBen - Commands Reference

Quick reference for building, installing, and running the BigBen React Native app.

---

## 📦 Initial Project Setup

### Create New Expo Project (one-time)
```bash
npx create-expo-app@latest BigBen --template blank-typescript
```

### Install Core Dependencies (one-time)
```bash
cd BigBen
npx expo install expo-sqlite @react-native-async-storage/async-storage \
  @react-navigation/native @react-navigation/native-stack \
  react-native-screens react-native-safe-area-context react-native-gesture-handler
```

### Install React Navigation
```bash
npm install @react-navigation/native @react-navigation/native-stack
```

---

## 🚀 Development (Expo Go)

### Start Dev Server
```bash
npx expo start
```

### Start with Cache Cleared
```bash
npx expo start --clear
```

### Offline Mode (no Expo Go download)
```bash
npx expo start --offline
```

### Run on Android
```bash
npx expo start --android
```

### Run on iOS
```bash
npx expo start --ios
```

---

## 🔨 Building APK (Standalone)

### Step 1: Prebuild (creates native android/ folder)
```bash
npx expo prebuild --platform android
```

### Step 2a: Build Debug APK with Gradle
```bash
cd android
./gradlew assembleDebug
```
**Output:** `android/app/build/outputs/apk/debug/app-debug.apk`

### Step 2b: Build Release APK (production)
```bash
cd android
./gradlew assembleRelease
```
**Output:** `android/app/build/outputs/apk/release/app-release.apk`

## 📲 Install on Phone via ADB

### Check Connected Devices
```bash
adb devices
```

### Install Debug APK
```bash
# From project root:
adb install android/app/build/outputs/apk/debug/app-debug.apk

# From inside android/ directory:
adb install app/build/outputs/apk/debug/app-debug.apk
```

### Build + Install in One Step
```bash
cd android
./gradlew installDebug
```

### Uninstall App
```bash
adb uninstall com.anonymous.bigben
```

### Force Reinstall (replace existing)
```bash
adb install -r android/app/build/outputs/apk/debug/app-debug.apk
```

## 🧹 Cleanup & Troubleshooting

### Clear Metro Bundler Cache
```bash
npx expo start --clear
```

### Reset Node Modules (if things break)
```bash
rm -rf node_modules package-lock.json
npm install
```

### Clean Gradle Build (if build fails)
```bash
cd android
./gradlew clean
```

### Clean Everything (deep reset)
```bash
cd android
./gradlew clean
cd ..
rm -rf node_modules package-lock.json
npm install
npx expo prebuild --clean
```

## ⚙️ Common Issues & Fixes

### "Read ETIMEDOUT" error
- Network issue, try `--offline` flag or switch networks

### "No Android device found"
- Enable USB debugging on phone
- Run `adb devices` to verify connection
- Try a different USB cable

### "Build failed" after dependency change
```bash
cd android
./gradlew clean
cd ..
npx expo prebuild --clean
cd android && ./gradlew assembleDebug
```

### "Package conflicts" / "Duplicate modules"
```bash
rm -rf node_modules
npm install
npx expo prebuild --clean
```

## 📋 Phone Setup (one-time)

To install APKs from your computer:

1. **Enable Developer Options** on phone
   - Settings → About Phone → Tap "Build Number" 7 times

2. **Enable USB Debugging**
   - Settings → Developer Options → USB Debugging → ON

3. **Allow Install from Unknown Sources**
   - Settings → Security → Install unknown apps → Allow

4. **Verify connection**
   ```bash
   adb devices
   ```
   Should show your device listed
